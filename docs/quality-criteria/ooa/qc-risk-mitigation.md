# Quality Criteria for Risk & Mitigation

This document provides a template for defining quality criteria for each risk and mitigation action.


## Instructions
- For each risk, specify quality criteria based on FURPS (Functionality, Usability, Reliability, Performance, Supportability) and relevant KPIs.
- Reference docs/furps.md and docs/kpi.md for details.

---
Update this file as you define new risks and their quality criteria.
